## üìñ About the Project
**DETI Dungeons** is a turn-based strategy RPG built from scratch using **Python**. 
The main goal of this project is to apply advanced programming concepts such as **Object-Oriented Programming (OOP)**, state management, and algorithm logic (damage calculation, turn flow) without relying on heavy game engines.

While the initial version runs on the CLI (Command Line Interface) to ensure solid logic, the final version will feature a GUI built with **Pygame**.
